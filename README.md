# The Dicee Roller Game 🎲

A simple, interactive browser game that generates random numbers to simulate dice rolls and declares a winner. 

## Description
This project demonstrates foundational JavaScript logic and Document Object Model (DOM) manipulation. Upon refreshing the page, the application dynamically generates random numbers, updates the corresponding dice image sources, and instantly evaluates the win-state to declare Player 1 or Player 2 as the winner (or a draw).

## Features
- Dynamic DOM manipulation
- Randomized algorithmic logic
- Instant win-state evaluation

## Technologies Used
- **Frontend:** HTML5, CSS3
- **Logic:** JavaScript (ES6+)

## How to Run Locally
1. Clone this repository to your local machine.
2. Open the project folder.
3. Double-click the `index.html` file to open it in your default web browser.
4. Refresh the page to roll the dice!
